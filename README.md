# Projeto_Clube_do_livro_2025


![]( https://imgur.com/YDsOl6b.gif )
## Projeto

Desenvolvido durante o curso Fullstack da [Academia do Programador](https://www.academiadoprogramador.net) 2025

## Entradas 

* O programa conta com quatro menus diferentes: o menu principal, o menu de revistas, o menu de amigos e o menu de empréstimos.

* Cada menu possui seu próprio sistema de entrada de dados. A principal forma de entrada ocorre através de comparações usando o operador ==, onde o usuário informa uma opção, e o programa compara essa entrada para determinar qual ação executar.

* No menu principal, o usuário escolhe para qual uma das telas deseja navegar: Revistas, Amigos ou Empréstimos.

* Dentro de cada tela (menu específico), o usuário pode selecionar diferentes opções, como cadastrar, editar, excluir ou visualizar registros, de acordo com a funcionalidade daquele menu.



## Detalhes 

Detalhes
Este projeto conta com um sistema de cadastro de registros, sendo eles:

* Novos amigos

* Novos empréstimos

* Novas caixas

* Novas revistas

* Cada um desses registros possui a possibilidade de realizar as seguintes ações básicas:

* Criar (Cadastrar)

* Visualizar (Listar)

* Editar (Alterar informações)

* Excluir (Remover do sistema)



# funcionalidades

## Validação de entradas
* O sistema verifica se as entradas do usuário são válidas

* Se a entrada estiver errada, permite tentar novamente até acertar

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
